# Travel Blog

## Dependencies

- [React] - required v17+. 

## Installation

Requires [Node.js](https://nodejs.org/) v15+ to run.

Install the dependencies and devDependencies.

```sh
npm i
```


## Development

Start production build:

```sh
npm run build:prod
```

Start development build:

```sh
npm run build:dev
```

Start eslint:

```sh
npm run lint
```

Start eslint fix:

```sh
npm run lint:fix
```

Start development server:

```sh
npm run dev:server
```


   [React]: <https://ru.reactjs.org/>
